# AIToolsRadar

Compare and discover 40+ AI tools.

[![Live Demo](https://img.shields.io/badge/Live_Demo-aitoolsradar.vercel.app-blue?style=for-the-badge)](https://aitoolsradar-zeta.vercel.app)

[Live Demo](https://aitoolsradar-zeta.vercel.app) | [Report Bug](https://github.com/keyflowcoreg/aitoolsradar/issues)

## Features

- 40+ AI tools reviewed across 6 categories (code, writing, design, marketing, data, automation)
- Side-by-side comparison tables
- Individual tool review pages with ratings
- SEO-optimized with dynamic sitemap and server-rendered pages
- Newsletter signup for weekly AI tool updates

## Tech Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Framer Motion

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4004](http://localhost:4004) to view it locally.
